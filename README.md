## Featured Projects

### [ST-GAT Earthquake](https://github.com/YukiEuler/st-gat-earthquake)

A spatio-temporal deep learning framework for microseismic magnitude prediction.

The project combines Graph Attention Networks and LSTM models to learn spatial and temporal patterns from earthquake sequences. It also implements deep ensembles and post-hoc calibration for uncertainty estimation.

**Highlights:** PyTorch, PyTorch Geometric, GAT, LSTM, uncertainty calibration, ablation studies, multi-resolution forecasting.

---

### [GovChain](https://github.com/YukiEuler/stellar-govchain)

A smart-contract system designed to improve transparency and accountability in government budget disbursement.

GovChain models processes such as budget allocation, payment proposals, multi-signature approval, vendor validation, and fund disbursement using Soroban smart contracts on the Stellar network.

**Highlights:** Rust, Soroban, Stellar, smart contracts, multisignature authorization, domain-driven system design.

---

### [NetraSiaga](https://github.com/YukiEuler/netra-siaga)

A real-time AI system for detecting driver drowsiness through a browser camera or uploaded video.

The application classifies driver conditions such as focused driving, talking, yawning, and microsleep, then activates an audio warning when dangerous behavior is detected.

**Highlights:** MobileNetV3, LSTM, PyTorch, ONNX, Streamlit, WebRTC, real-time inference.

---

### [JawaSeek](https://github.com/YukiEuler/JawaSeek)

A cross-lingual information retrieval application for searching Javanese Wikipedia articles using Indonesian-language queries.

The system supports traditional and neural retrieval methods, including dictionary-based TF-IDF, multilingual BERT, LaBSE, and fine-tuned language models.

**Highlights:** NLP, semantic search, mBERT, LaBSE, TF-IDF, Streamlit, Indonesian–Javanese language technology.

---

### [GeoFarm](https://github.com/YukiEuler/GeoFarm)

A Flutter-based application developed as a HackFest 2024 submission.

**Highlights:** Dart, Flutter, mobile application development, collaborative product development.

---

### [Skripsi LaTeX IF Undip](https://github.com/YukiEuler/Skripsi-Latex-IF-Undip)

A modular LaTeX template for Informatics undergraduate theses at Diponegoro University.

It supports APA 7th edition citations, automated numbering, and an organized document structure for academic writing.

**Highlights:** LaTeX, BibLaTeX, academic tooling, document automation.
